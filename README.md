# Personal Site — Srijit Banerjee

A personal portfolio website built completely from scratch using vanilla HTML, CSS, and JavaScript. It serves as my own corner on the internet to showcase my coding projects, STEM background, hobbies, and digital presence.

---

## 🛠️ Built With

* **HTML5** — Semantic markup and page layout
* **CSS3** — Custom layout, color themes, and animations
* **JavaScript** — DOM manipulation for interactive widgets and mini-games

---

## 🌟 What's Inside

* **About & Hero:** Short intro featuring a dynamic typing title and interactive manager card.
* **Favorites & Hobbies:** Ranked showcases for movies, series, and games.
* **Polyglot Vault & Honors:** Language learning metrics and verified STEM/Olympiad accomplishments.
* **Interactive Corner:** Interactive pitch minigame, quote generator, and tactical chess puzzle board.
* **Digital Footprint:** Direct links to social media accounts and math blog.
